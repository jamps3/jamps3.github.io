## Selected Projects

### [Quantifile](https://github.com/jamps3/Quantifile)

A disk space visualization tool inspired by the classic SpaceMonger application. Quantifile scans directories and displays files and folders as an interactive treemap, making it easier to identify large files and understand disk usage visually.

**Highlights:**

- Interactive treemap UI where rectangle size represents file or folder size
- Recursive directory scanning with background processing to keep the UI responsive
- Keyboard navigation, zooming, context menus, bookmarks, search, filtering, and SVG export
- Permission-error handling with visible access-denied placeholders and non-disruptive log messages
- Standalone Windows ZIP release in addition to Python source usage

**Tech stack:** Python · Tkinter · File systems · Treemap visualization · Background scanning · SVG export · Desktop GUI
